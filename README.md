# Brain_Tumor_Classification
Dự án nghiên cứu các giải pháp kết hợp giữa Computer Vision và Deep Learning cho bài toán Brain Tumor Classification (Phân loại u não) trên ảnh MRI 2D, qua đó nhằm đề xuất một giải pháp tối ưu nhất cho bài toán Phân loại u não trên ảnh MRI 2D - có hiệu suất tốt và khả năng ứng dụng cao trong thực tế lĩnh vực y tế.
## Link dataset sử dụng trong nghiên cứu:
- Bộ dữ liệu train: https://data.mendeley.com/datasets/zwr4ntf94j/4
- Bộ dữ liệu kiểm thử: https://huggingface.co/datasets/Simezu/brain-tumour-MRI-scan
## Hướng dẫn thực thi mã nguồn:
Toàn bộ code của dự án được thực thi trên môi trường GPU T4 của Google Colab, kết hợp với hạ tầng lưu trữ của Google Drive.
Để thực thi code, hãy thực hiện các bước sau:
- 1. Chuẩn bị môi trường: Chuẩn bị môi trường Google Colaboratory, phần cứng lưu trữ Google Drive.
- 2. Chuẩn bị dữ liệu: Tải về hai bộ dữ liệu train và test, lưu vào drive.
- 3. Thay đổi các đường dẫn trong các file code về đúng với đường dẫn trên Google Drive của bạn và thực thi code.
